# SharePoint Discovery App

## Description

Enter a SharePoint Online URL and valid credentials. Click on "Get Sites" to show the sites and sub-sites. From there, you can show all of the Lists within each site, and then all of the Views and Fields within each List. You can also search by List Name (i.e. Title) or List GUID.
